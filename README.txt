# 项目名称：C语言中文头文件

## 简介

本项目提供了一个用于娱乐性质的C语言头文件，它允许使用中文关键字进行编程。请注意，这个头文件不建议在生产环境中使用。

## 使用说明

1. 将 `stvcn.h`  头文件包含到你的C语言项目中。
2. 使用中文关键字进行编程，例如使用 `主函数` 代替 `main`，`整型` 代替 `int` 等。
